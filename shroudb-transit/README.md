# ShroudbTransit Python Client

Python client for the [ShroudbTransit](https://github.com/shroudb/shroudb-transit) Encryption-as-a-service.

## Install

```bash
pip install shroudb-transit
```

## Quick Start

```python
import asyncio
from shroudb_transit import ShroudbTransitClient

async def main():
    client = await ShroudbTransitClient.connect("shroudb-transit://localhost")

    # Issue a credential
    result = await client.issue("my-keyspace", ttl_secs=3600)
    print(result.credential_id, result.token)

    # Verify it
    verified = await client.verify("my-keyspace", result.token)
    print(verified.state)  # "active"

    await client.close()

asyncio.run(main())
```

## Connection URI

```
shroudb-transit://[token@]host[:port][/keyspace]
shroudb-transit+tls://[token@]host[:port][/keyspace]
```

Examples:
- `shroudb-transit://localhost` — plain TCP, default port 6499
- `shroudb-transit+tls://prod.example.com` — TLS
- `shroudb-transit://mytoken@localhost:6499/sessions` — auth + keyspace

## Connection Pool

The client maintains an internal connection pool. Tune it at connect time:

```python
client = await ShroudbTransitClient.connect("shroudb-transit://localhost", max_idle=8, max_open=32)
```

## Commands

- `client.decrypt(...)` — Decrypt ciphertext using the embedded key version
- `client.encrypt(...)` — Encrypt plaintext with the active key version
- `client.generate_data_key(...)` — Generate a data encryption key (envelope encryption pattern)
- `client.health(...)` — Check server health
- `client.key_info(...)` — Get keyring metadata and key version information
- `client.rewrap(...)` — Re-encrypt ciphertext with the current active key version
- `client.rotate(...)` — Rotate the keyring to a new key version
- `client.sign(...)` — Create a detached signature
- `client.verify_signature(...)` — Verify a detached signature


## Real-time Subscriptions

```python
async with await ShroudbTransitClient.connect("shroudb-transit://localhost") as client:
    async with await client.subscribe("keyspace:tokens") as sub:
        async for event in sub:
            print(event.event_type, event.keyspace, event.detail, event.timestamp)
```

Use `"*"` to subscribe to all events. Close the subscription (or exit the
`async with` block) to stop receiving events.
