# ShroudbAuth Python Client

Python client for the [ShroudbAuth](https://github.com/shroudb/shroudb-auth) Authentication service.

## Install

```bash
pip install shroudb-auth
```

## Quick Start

```python
import asyncio
from shroudb_auth import ShroudbAuthClient

async def main():
    async with ShroudbAuthClient("http://localhost:4001") as client:
        result = await client.change_password("...", "...")
        print(result)

asyncio.run(main())
```

## Constructor

```python
client = ShroudbAuthClient(
    "http://localhost:4001",
    keyspace="my-app",  # default keyspace for all requests
    timeout=30.0,       # request timeout in seconds
)
```

## Token Management

After calling an endpoint that returns tokens, the client automatically stores
the access and refresh tokens. Subsequent calls to authenticated endpoints use
these tokens automatically.

You can also manage tokens manually:

```python
client.access_token = "eyJ..."
client.refresh_token = "rt_..."
```

## Methods

- `client.change_password(...)` — Change password for the currently authenticated user
- `client.forgot_password(...)` — Request a password reset token (always returns 200 to prevent enumeration)
- `client.health(...)` — Health check endpoint
- `client.jwks(...)` — Public JSON Web Key Set for verifying access tokens
- `client.login(...)` — Authenticate a user and receive access + refresh tokens
- `client.logout(...)` — Revoke the current refresh token family and clear cookies
- `client.logout_all(...)` — Revoke all refresh token families for a user
- `client.refresh(...)` — Exchange a refresh token for new access + refresh tokens
- `client.reset_password(...)` — Reset password using a single-use reset token (revoked after use)
- `client.session(...)` — Validate current session and return user info
- `client.sessions(...)` — List active sessions (refresh token families) for the authenticated user
- `client.signup(...)` — Register a new user and receive access + refresh tokens
