# ShroudbCourier Python Client

Python client for the [ShroudbCourier](https://github.com/shroudb/shroudb-courier) Secure notification delivery pipeline.

## Install

```bash
pip install shroudb-courier
```

## Quick Start

```python
import asyncio
from shroudb_courier import ShroudbCourierClient

async def main():
    client = await ShroudbCourierClient.connect("shroudb-courier://localhost")

    # Issue a credential
    result = await client.issue("my-keyspace", ttl_secs=3600)
    print(result.credential_id, result.token)

    # Verify it
    verified = await client.verify("my-keyspace", result.token)
    print(verified.state)  # "active"

    await client.close()

asyncio.run(main())
```

## Connection URI

```
shroudb-courier://[token@]host[:port][/keyspace]
shroudb-courier+tls://[token@]host[:port][/keyspace]
```

Examples:
- `shroudb-courier://localhost` — plain TCP, default port 6999
- `shroudb-courier+tls://prod.example.com` — TLS
- `shroudb-courier://mytoken@localhost:6999/sessions` — auth + keyspace

## Connection Pool

The client maintains an internal connection pool. Tune it at connect time:

```python
client = await ShroudbCourierClient.connect("shroudb-courier://localhost", max_idle=8, max_open=32)
```

## Commands

- `client.auth(...)` — Authenticate the connection
- `client.channel_info(...)` — Get subscriber count for a WebSocket channel
- `client.channel_list(...)` — List all active WebSocket channels
- `client.connections(...)` — Get total WebSocket connections
- `client.deliver(...)` — Deliver a notification (decrypts recipient, renders template, sends via adapter)
- `client.health(...)` — Check server health
- `client.template_info(...)` — Get information about a specific template
- `client.template_list(...)` — List all loaded templates
- `client.template_reload(...)` — Reload templates from disk


## Real-time Subscriptions

```python
async with await ShroudbCourierClient.connect("shroudb-courier://localhost") as client:
    async with await client.subscribe("keyspace:tokens") as sub:
        async for event in sub:
            print(event.event_type, event.keyspace, event.detail, event.timestamp)
```

Use `"*"` to subscribe to all events. Close the subscription (or exit the
`async with` block) to stop receiving events.

## Context Manager

```python
async with await ShroudbCourierClient.connect("shroudb-courier://localhost") as client:
    result = await client.issue("tokens")
```
