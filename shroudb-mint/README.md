# ShroudbMint Python Client

Python client for the [ShroudbMint](https://github.com/shroudb/shroudb-mint) Lightweight internal Certificate Authority.

## Install

```bash
pip install shroudb-mint
```

## Quick Start

```python
import asyncio
from shroudb_mint import ShroudbMintClient

async def main():
    client = await ShroudbMintClient.connect("shroudb-mint://localhost")

    # Issue a credential
    result = await client.issue("my-keyspace", ttl_secs=3600)
    print(result.credential_id, result.token)

    # Verify it
    verified = await client.verify("my-keyspace", result.token)
    print(verified.state)  # "active"

    await client.close()

asyncio.run(main())
```

## Connection URI

```
shroudb-mint://[token@]host[:port][/keyspace]
shroudb-mint+tls://[token@]host[:port][/keyspace]
```

Examples:
- `shroudb-mint://localhost` — plain TCP, default port 6699
- `shroudb-mint+tls://prod.example.com` — TLS
- `shroudb-mint://mytoken@localhost:6699/sessions` — auth + keyspace

## Connection Pool

The client maintains an internal connection pool. Tune it at connect time:

```python
client = await ShroudbMintClient.connect("shroudb-mint://localhost", max_idle=8, max_open=32)
```

## Commands

- `client.auth(...)` — Authenticate the connection
- `client.ca_create(...)` — Create a new Certificate Authority
- `client.ca_export(...)` — Export the CA's public certificate
- `client.ca_info(...)` — Get information about a CA
- `client.ca_list(...)` — List all CAs
- `client.ca_rotate(...)` — Rotate the CA's signing key
- `client.crl_info(...)` — Get CRL information for a CA
- `client.health(...)` — Check server health
- `client.inspect(...)` — Inspect a certificate
- `client.issue(...)` — Issue a new certificate
- `client.issue_from_csr(...)` — Issue a certificate from a CSR
- `client.list_certs(...)` — List certificates for a CA
- `client.renew(...)` — Renew a certificate
- `client.revoke(...)` — Revoke a certificate


## Real-time Subscriptions

```python
async with await ShroudbMintClient.connect("shroudb-mint://localhost") as client:
    async with await client.subscribe("keyspace:tokens") as sub:
        async for event in sub:
            print(event.event_type, event.keyspace, event.detail, event.timestamp)
```

Use `"*"` to subscribe to all events. Close the subscription (or exit the
`async with` block) to stop receiving events.

async with await ShroudbMintClient.connect("shroudb-mint://localhost") as client:
