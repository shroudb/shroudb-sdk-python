# ShroudbKeep Python Client

Python client for the [ShroudbKeep](https://github.com/shroudb/shroudb-keep) Secrets manager.

## Install

```bash
pip install shroudb-keep
```

## Quick Start

```python
import asyncio
from shroudb_keep import ShroudbKeepClient

async def main():
    client = await ShroudbKeepClient.connect("shroudb-keep://localhost")

    # Issue a credential
    result = await client.issue("my-keyspace", ttl_secs=3600)
    print(result.credential_id, result.token)

    # Verify it
    verified = await client.verify("my-keyspace", result.token)
    print(verified.state)  # "active"

    await client.close()

asyncio.run(main())
```

## Connection URI

```
shroudb-keep://[token@]host[:port][/keyspace]
shroudb-keep+tls://[token@]host[:port][/keyspace]
```

Examples:
- `shroudb-keep://localhost` — plain TCP, default port 6899
- `shroudb-keep+tls://prod.example.com` — TLS
- `shroudb-keep://mytoken@localhost:6899/sessions` — auth + keyspace

## Connection Pool

The client maintains an internal connection pool. Tune it at connect time:

```python
client = await ShroudbKeepClient.connect("shroudb-keep://localhost", max_idle=8, max_open=32)
```

## Commands

- `client.auth(...)` — Authenticate the connection
- `client.delete(...)` — Soft-delete a secret
- `client.get(...)` — Retrieve a secret (latest or specific version)
- `client.health(...)` — Check server health
- `client.list(...)` — List secret paths matching a prefix
- `client.put(...)` — Store a secret (creates a new version)
- `client.rotate(...)` — Re-encrypt the latest version under the current key
- `client.versions(...)` — Show version history for a secret


## Real-time Subscriptions

```python
async with await ShroudbKeepClient.connect("shroudb-keep://localhost") as client:
    async with await client.subscribe("keyspace:tokens") as sub:
        async for event in sub:
            print(event.event_type, event.keyspace, event.detail, event.timestamp)
```

Use `"*"` to subscribe to all events. Close the subscription (or exit the
`async with` block) to stop receiving events.

## Context Manager

```python
async with await ShroudbKeepClient.connect("shroudb-keep://localhost") as client:
    result = await client.issue("tokens")
```
