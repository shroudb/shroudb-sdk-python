# ShroudbSentry Python Client

Python client for the [ShroudbSentry](https://github.com/shroudb/shroudb-sentry) Policy-based authorization engine.

## Install

```bash
pip install shroudb-sentry
```

## Quick Start

```python
import asyncio
from shroudb_sentry import ShroudbSentryClient

async def main():
    client = await ShroudbSentryClient.connect("shroudb-sentry://localhost")

    # Issue a credential
    result = await client.issue("my-keyspace", ttl_secs=3600)
    print(result.credential_id, result.token)

    # Verify it
    verified = await client.verify("my-keyspace", result.token)
    print(verified.state)  # "active"

    await client.close()

asyncio.run(main())
```

## Connection URI

```
shroudb-sentry://[token@]host[:port][/keyspace]
shroudb-sentry+tls://[token@]host[:port][/keyspace]
```

Examples:
- `shroudb-sentry://localhost` — plain TCP, default port 6799
- `shroudb-sentry+tls://prod.example.com` — TLS
- `shroudb-sentry://mytoken@localhost:6799/sessions` — auth + keyspace

## Connection Pool

The client maintains an internal connection pool. Tune it at connect time:

```python
client = await ShroudbSentryClient.connect("shroudb-sentry://localhost", max_idle=8, max_open=32)
```

## Commands

- `client.auth(...)` — Authenticate the connection
- `client.evaluate(...)` — Evaluate an authorization request against loaded policies
- `client.health(...)` — Check server health
- `client.key_info(...)` — Get signing key information
- `client.key_rotate(...)` — Rotate the signing key used for JWT decisions
- `client.policy_info(...)` — Get information about a specific policy
- `client.policy_list(...)` — List all loaded policies
- `client.policy_reload(...)` — Reload policies from disk


## Real-time Subscriptions

```python
async with await ShroudbSentryClient.connect("shroudb-sentry://localhost") as client:
    async with await client.subscribe("keyspace:tokens") as sub:
        async for event in sub:
            print(event.event_type, event.keyspace, event.detail, event.timestamp)
```

Use `"*"` to subscribe to all events. Close the subscription (or exit the
`async with` block) to stop receiving events.

## Context Manager

async with await ShroudbSentryClient.connect("shroudb-sentry://localhost") as client:
