# Shroudb Python Client

Python client for the [Shroudb](https://github.com/shroudb/shroudb) Credential management server.

## Install

```bash
pip install shroudb
```

## Quick Start

```python
import asyncio
from shroudb import ShroudbClient

async def main():
    client = await ShroudbClient.connect("shroudb://localhost")

    # Issue a credential
    result = await client.issue("my-keyspace", ttl_secs=3600)
    print(result.credential_id, result.token)

    # Verify it
    verified = await client.verify("my-keyspace", result.token)
    print(verified.state)  # "active"

    await client.close()

asyncio.run(main())
```

## Connection URI

```
shroudb://[token@]host[:port][/keyspace]
shroudb+tls://[token@]host[:port][/keyspace]
```

Examples:
- `shroudb://localhost` — plain TCP, default port 6399
- `shroudb+tls://prod.example.com` — TLS
- `shroudb://mytoken@localhost:6399/sessions` — auth + keyspace

## Connection Pool

The client maintains an internal connection pool. Tune it at connect time:

```python
client = await ShroudbClient.connect("shroudb://localhost", max_idle=8, max_open=32)
```

## Commands

- `client.auth(...)` — Authenticate the current connection
- `client.config_get(...)` — Retrieve a runtime configuration value
- `client.config_set(...)` — Set a runtime configuration value
- `client.health(...)` — Check server or keyspace health
- `client.inspect(...)` — Retrieve full details about a credential
- `client.issue(...)` — Issue a new credential in the given keyspace
- `client.jwks(...)` — Return the JSON Web Key Set for a JWT keyspace
- `client.keys(...)` — List credential IDs with optional filtering and pagination
- `client.keystate(...)` — Show the current key ring state for a keyspace
- `client.password_change(...)` — Change a user's password (requires old password)
- `client.password_import(...)` — Import a pre-hashed password for migration from another system (argon2, bcrypt, scrypt)
- `client.password_set(...)` — Set a password for a user in a password keyspace
- `client.password_verify(...)` — Verify a user's password
- `client.refresh(...)` — Exchange a refresh token for a new one
- `client.revoke(...)` — Revoke a credential by ID
- `client.revoke_bulk(...)` — Bulk-revoke multiple credentials
- `client.revoke_family(...)` — Revoke all credentials in a refresh token family
- `client.rotate(...)` — Trigger signing key rotation for a keyspace
- `client.schema(...)` — Display the metadata schema for a keyspace
- `client.suspend(...)` — Temporarily suspend a credential
- `client.unsuspend(...)` — Reactivate a previously suspended credential
- `client.update(...)` — Update metadata on an existing credential
- `client.verify(...)` — Verify a credential (JWT, API key, or HMAC signature)


## Real-time Subscriptions

```python
async with await ShroudbClient.connect("shroudb://localhost") as client:
    async with await client.subscribe("keyspace:tokens") as sub:
        async for event in sub:
            print(event.event_type, event.keyspace, event.detail, event.timestamp)
```

Use `"*"` to subscribe to all events. Close the subscription (or exit the
`async with` block) to stop receiving events.

## Context Manager

async with await ShroudbClient.connect("shroudb://localhost") as client:
    result = await client.issue("tokens")
