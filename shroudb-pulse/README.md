# ShroudbPulse Python Client

Python client for the [ShroudbPulse](https://github.com/shroudb/shroudb-pulse) Observability plane for unified audit event streaming.

## Install

```bash
pip install shroudb-pulse
```

## Quick Start

```python
import asyncio
from shroudb_pulse import ShroudbPulseClient

async def main():
    client = await ShroudbPulseClient.connect("shroudb-pulse://localhost")

    # Issue a credential
    result = await client.issue("my-keyspace", ttl_secs=3600)
    print(result.credential_id, result.token)

    # Verify it
    verified = await client.verify("my-keyspace", result.token)
    print(verified.state)  # "active"

    await client.close()

asyncio.run(main())
```

## Connection URI

```
shroudb-pulse://[token@]host[:port][/keyspace]
shroudb-pulse+tls://[token@]host[:port][/keyspace]
```

Examples:
- `shroudb-pulse://localhost` — plain TCP, default port 7099
- `shroudb-pulse+tls://prod.example.com` — TLS
- `shroudb-pulse://mytoken@localhost:7099/sessions` — auth + keyspace

## Connection Pool

The client maintains an internal connection pool. Tune it at connect time:

```python
client = await ShroudbPulseClient.connect("shroudb-pulse://localhost", max_idle=8, max_open=32)
```

## Commands

- `client.actors(...)` — Most active actors
- `client.auth(...)` — Authenticate the connection
- `client.count(...)` — Count events matching filter arguments
- `client.errors(...)` — Per-operation error rates
- `client.health(...)` — Check server health
- `client.hotspots(...)` — Find hotspot resources with highest activity
- `client.ingest(...)` — Ingest a single audit event
- `client.ingest_batch(...)` — Ingest a batch of audit events
- `client.query(...)` — Query events with filter arguments
- `client.source_list(...)` — List configured event sources
- `client.source_status(...)` — Show per-source ingestion statistics


## Real-time Subscriptions

```python
async with await ShroudbPulseClient.connect("shroudb-pulse://localhost") as client:
    async with await client.subscribe("keyspace:tokens") as sub:
        async for event in sub:
            print(event.event_type, event.keyspace, event.detail, event.timestamp)
```

Use `"*"` to subscribe to all events. Close the subscription (or exit the
`async with` block) to stop receiving events.
